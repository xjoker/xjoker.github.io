碎碎念

Remote Desktop Manager 是Windows和MAC平台下的一个多功能远程工具，可以同时远程Linux和Windows，而且拥有很多更加强大的功能。
目前网上破解方法是序列号加上屏蔽验证域名，但是这样搞得在web调试时切换HOST很容易就激活失败。
所以决定逆向一下，看下去除联网验证。

项目分析

Remote Desktop Manager(RDM)是使用.NET写成的，将程序拖入dnSpy中。
启动调试

可以看见激活页面

dnSpy中搜索关键字

可以看见方法就叫做 RegistrationManager

跟踪几个调用后，找到了注册窗体。也找到了提交验证的地方。
可见代码只是进行了简单的混淆，不妨碍分析工作。

这里可以看见传入了两个参数就是Email和序列号

方法根据 RDMOManager.SendRegistrationEnterprise 返回的结果来定义是否通过验证

修改代码

跟踪后发现其实很简单，只要修改 “ca41c2f6529c2be524a8575ca80918992” 方法的返回永远为True即可

测试

正常激活成功，联网验证搞定.
